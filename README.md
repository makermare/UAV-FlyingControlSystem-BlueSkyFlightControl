#### 介绍
# BlueSkyFlightControl
算法上基于的非线性卡尔曼滤波、梯度下降预估最小二乘法的改版，
支持导航、光流、激光定高。

本项目相对原始天穹飞控，有如下主要变化：
1. 主控芯片换为 STM32H743VI。
2. 增加了一些传感器驱动和光流、激光高度计传感器。
3. 增加了外部控制飞行的Python脚本，支持视觉识别。
4. 增加了代码的文档。
